# Rule-Based DOM Inference — Benchmark Results

Run `npm run test:impl -- rule-based` to populate this directory with JSON reports.

## Expected files after a run

- `quick-results.json` — quick benchmark (5 instances per form)
- `full-results.json` — full benchmark (50 instances per form)
- `domain-breakdown.json` — per-domain metrics
- `report.html` — human-readable HTML report
